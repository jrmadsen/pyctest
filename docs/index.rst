Welcome to PyCTest's documentation!
===================================

.. title:: PyCTest

`PyCTest <https://github.com/jrmadsen/pyctest.git>`_ is an open-source
Python package for select bindings to CMake, CTest, and CPack.

Features
========

* Directly run `cmake`, `ctest`, `cpack`.
* Dynamically generate testing and submit to CDash testing dashboard
